### 🔹 앱 소개

**한땀한땀**은 AI 인증 시스템을 통해 공정하고 투명한 챌린지를 제공하는 플랫폼입니다.

사진 인증을 AI가 자동 검증하여 부정행위를 방지하고,

Apple Watch 연동으로 운동 기록을 실시간으로 인증할 수 있습니다.

***AI와 함께하는 새로운 습관, 한땀한땀에서 시작하세요!***

### 🔹 자주 묻는 질문

**Q. 회원가입은 어떻게 하나요?**

→ Apple, Kakao, Naver 로그인을 통해 간편하게 가입할 수 있습니다.

**Q. 챌린지는 어떻게 시작하나요?**

→ 원하는 주제의 챌린지를 개설하거나 기존 그룹에 참여하면 됩니다.

**Q. AI 인증은 어떻게 작동하나요?**

→ 사진·영상 업로드 시 AI가 자동으로 판별하여 인증 결과를 제공합니다.

**Q. Apple Watch는 어떻게 연동하나요?**

→ 장기 이벤트 미션의 경우 HealthKit 권한을 허용하면 Apple Watch 기록이 자동 인증됩니다.
→ 단기 이벤트 미션의 경우 애플워치 앱을 통해 실시간 게임 형태로 기록을 인증합니다.

**Q. 다른 사용자와는 어떻게 소통하나요?**

→ 인증 피드에 댓글과 좋아요를 남기거나 그룹 채팅에서 대화할 수 있습니다.
